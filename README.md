## 🌐 UC20: Angular Frontend for Quantity


## 🧠 Project Overview

The **UC20 Frontend for Quantity Measurement Application** is developed using **Angular Framework** to build a scalable, modular, and dynamic user interface.

This frontend interacts with the backend to perform **measurement operations, unit conversions, authentication, and user interactions**.

Implemented using:

- ✅ Angular (Standalone Components & Routing)  
- ✅ TypeScript (Strongly Typed JavaScript)  
- ✅ HTML5 & CSS3  
- ✅ Angular Services & Dependency Injection  
- ✅ HTTP Client for API Integration  
- ✅ Reactive and Dynamic UI Rendering
